# VinBigData Chest X-ray Detection

## Описание

В работе рассматривается задача детекции патологий грудной клетки на датасете `VinBigData Chest X-ray Abnormalities Detection` с использованием `YOLO11`.

Основной фокус работы — улучшение качества детекции редких патологий и анализ влияния различных методов борьбы с дисбалансом классов.

---

## Что было сделано

### Baseline

- Обучена baseline YOLO11 модель.
- Выполнено финальное обучение на `train + val`.
- Проведена оценка на `test`.
- Выполнен анализ ошибок по классам (`FP`, `FN`).

### Аугментации

Для grayscale X-ray изображений были отключены нерелевантные HSV-аугментации (`hsv_h`, `hsv_s`, `hsv_v`).

Использовались аугментации, более подходящие для медицинских снимков:

- `translate`
- `scale`
- `fliplr`
- `mosaic`
- `mixup`

### Per-class Threshold Tuning

Для каждого класса отдельно были подобраны confidence thresholds на `val`.

Подбор выполнялся с оптимизацией по `F2-score`, так как в медицинской задаче recall важнее precision: пропуск патологии обычно критичнее ложноположительного предсказания.

### Oversampling + CLAHE

- Выполнен oversampling редких классов.
- Использован CLAHE preprocessing для улучшения локального контраста изображений.

### Weighted Loss + CLAHE

- Использованы веса классов в loss function для компенсации дисбаланса данных.
- Дополнительно применён CLAHE preprocessing.

---

## Результаты

- Baseline модель показала хорошее качество для частых патологий, но плохо работала на редких классах.
- Per-class threshold tuning дал наиболее стабильное улучшение precision и общего баланса метрик.
- Oversampling + CLAHE улучшили recall части редких патологий, но не дали устойчивого роста общего качества.
- Наиболее успешным вариантом оказался `Weighted Loss + CLAHE`, показавший лучший средний `F1-score` и улучшение качества для ряда редких патологий.

---

## Выводы

Основной проблемой датасета оказался сильный дисбаланс классов и сложность локализации некоторых патологий.

Использование per-class thresholds, weighted loss и CLAHE позволило улучшить качество модели для редких патологий при сохранении хороших результатов на частых классах.
